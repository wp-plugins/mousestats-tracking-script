The official plugin to install MouseStats Tracking Script on WordPress-powered websites.

== Description ==

By installing this plugin, you no longer need to insert the tracking script directly in template files. The plugin will include MouseStats tracking script
in all your pages with minimum configuration. You only need to set your MouseStats Account ID in your WordPress panel.

== Installation ==

1. Upload `mousestats-tracking-script` by using the "plugin install" in your WordPress management console, or directly upload the folder into the `/wp-content/plugins/` folder on your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In "WordPress General Settings" menu, navigate to "MouseStats Tracking Script" and enter your MouseStats Account ID.
4. Configure/activate/disable/control your analytics in MouseStats Panel

== Frequently Asked Questions ==

= What I need to generate a heatmap, visitor playback, form analytics or surveys? =

To utilize this plugin, you need to create an account in MouseStats to get an account ID. You are able to create a free or paid account in MouseStats website.
Up to 1 million visitor survey responses are provided at no cost.
